- **Primary CTA**: Gradient button with shine effect on hover

- **Decorative orbs**: Subtle floating elements in corners

### CSS Classes Used

**Glassmorphism:**
- `backdrop-blur-xl` - Strong blur effect
- `bg-white/10` - 10% white transparency
- `border-white/20` - 20% white border

**Animations:**
- `animate-fade-in-up` - Fade and slide up
- `animate-float` - Floating motion
- `animation-delay-100/200/300/400` - Staggered timing

**Effects:**
- `drop-shadow-2xl` - Heavy text shadows
- `shadow-emerald-500/50` - Colored glow effects
- `hover:scale-105` - Subtle scale on hover
